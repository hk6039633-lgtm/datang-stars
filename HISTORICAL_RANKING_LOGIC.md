# 唐局历史影响榜逻辑说明

## 设计目标

《唐局》前台人物榜只保留一个核心榜单：**历史影响榜**。该榜单不再直接沿用原书出场次数（`mentions` / `appearanceCount`），而是基于人物在真实唐朝历史中的综合影响力进行人工校准评分。

## 核心字段

| 字段 | 含义 | 用途 |
|------|------|------|
| `appearanceCount` | 原书出场次数 | 后台参考，不直接用于前台榜单 |
| `historicalImportanceScore` | 历史影响力评分 | **历史影响榜唯一排序依据** |
| `visualPriority` | 是否优先制作专属视觉图 | 视觉制作排期参考 |
| `subCategory` | 二级分类 | 榜单标签展示 |
| `impactStatement` | 一句话影响说明 | 榜单展示（未填时以 summary 兜底） |

## 评分原则

评分综合考虑：

1. **历史地位**：是否处于时代转折节点（开国、盛世、中兴、衰落、灭亡）。
2. **政治影响**：是否制定重大国策、改革制度、改变权力结构。
3. **军事影响**：是否主导或扭转关键战争、叛乱、统一进程。
4. **文化影响**：是否在文学、思想、宗教等领域留下深远遗产。
5. **长期后果**：其决策或行为对唐朝乃至后世的影响深度与广度。

## 分数区间

| 区间 | 定义 | 代表人物 |
|------|------|----------|
| 95-100 | 决定唐朝历史走向的顶级人物 | 李世民、武则天、李隆基、安禄山 |
| 90-94 | 开国、盛唐、中兴、乱世转折中的关键人物 | 李渊、李治、郭子仪、朱温 |
| 80-89 | 重要皇帝、宰相、名将、权臣、文化巨匠 | 李纯、李忱、魏征、李靖、李白、杜甫 |
| 70-79 | 区域性重要人物、关键事件参与者、重要文臣武将 | 李亨、李豫、李适、韩愈、黄巢 |
| 60-69 | 有明确史事价值但影响范围较小的人物 | 李显、李旦、李忠、部分节度使 |
| 60 以下 | 普通出场人物或资料性人物 | 大量配角 |

## 榜单展示

人物榜页面 `/ranking` 只展示一个榜单：

- **排序**：`historicalImportanceScore` 从高到低。
- **筛选**：支持按 `primaryCategory` 分类筛选（如只看「皇帝/皇室」）。
- **字段**：排名、头像、人物姓名、主要身份、所属分类、历史影响力评分、核心标签、一句话影响说明。

## 已移除榜单

本轮已移除以下榜单：

1. 全书出场榜
2. 皇帝功绩榜
3. 皇帝在位顺序榜

如需查看皇帝，请在「皇帝/皇室」分类筛选下按历史影响力排序。

## 搜索与别名

搜索人物时同时检索以下字段：

- `name` / `displayName`
- `canonicalName`
- `templeName`
- `dynastyTitle`
- `formerTitle`
- `aliases`

确保李世民、李隆基、李治等核心人物可通过本名、庙号、历史称号、早期封号搜索到。
